# E-commerce Orders Analysis Insights

## Overview
This project analyzes e-commerce customer behavior, order performance, and product revenue using MySQL.

## Key Insights
- Completed orders generated the core business revenue.
- Electronics produced the highest revenue because of their higher prices.
- A few customers contributed more revenue than others, showing strong differences in customer lifetime value.
- Product and category analysis helps identify the strongest drivers of sales.
- Monthly revenue trends can support planning and forecasting.

## Skills Demonstrated
- Relational database design
- Multi-table joins
- Revenue calculations
- Aggregate analysis
- Business reporting with SQL
- Customer lifetime value analysis
