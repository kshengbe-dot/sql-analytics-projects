# Customer Sales Analysis Insights

## Project Overview
This project analyzes customer purchases, product performance, and revenue trends using MySQL.

## Key Insights
- Electronics generated the highest revenue, showing they are the strongest product category.
- High-ticket items such as laptops and phones contributed more revenue than low-cost items.
- Some customers spent significantly more than others, making them high-value customers.
- Revenue varied across cities, which could help guide location-based marketing.
- Monthly revenue analysis helps identify sales patterns and business performance over time.

## Skills Demonstrated
- Database creation
- Table relationships using foreign keys
- Data cleaning checks
- SQL joins
- Aggregation functions
- Business-focused SQL analysis
